    - 整体步骤
        - 使用html合理规划组件结构

        - 将数据和html结合
        - 用户事件处理
    - 注意Date（）函数的使用方法月份从0开始数

    绑定的函数会被删除
    - css书写较为长，防止使用过程中的重名
